# LAB X

AI-powered Multimodal Visual Medical Diagnosis Assistant using RAG.

## Features

- Upload Medical Scans (X-Ray, MRI, CT)
- Automated Modality Classification (Chest, Brain, Abdomen, Knee)
- Localized Grad-CAM Attention Heatmaps
- Google Gemini Support
- Llama 3 / Mistral Support
- FAISS Vector Database
- Retrieved Literature Context Viewer
- Automated Medical Diagnostics Synthesis
- Interactive Clinical Q&A chatbot

## Technology Stack

- Python
- Streamlit
- PyTorch (EfficientNet-B4 & Grad-CAM)
- FAISS
- Google Gemini
- Llama 3
- NumPy & Matplotlib

## Run

streamlit run app.py
